## About

  건강과 환경을 동시에, 지금 바로 줍깅 하세요!

## Feature
  1. 내 주변 줍깅 모임을 검색하고 참여할 수 있습니다
  2. 줍깅 모임을 개설 할 수 있습니다.
  3. 모임 별 실시간 채팅을 통해 소통할 수 있습니다.
  4. 일정을 한눈에 파악할 수 있습니다.
  5. 커뮤니티 페이지에서 후기를 공유를 하고 댓글로 소통할 수 있습니다.
  6. 줍줍 포인트를 적립해서 환경뱃지를 획득 할 수 있습니다.
